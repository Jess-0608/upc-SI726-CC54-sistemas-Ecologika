Planes Flexibles

Puedes elegir el plan que mejor se acomode a tus necesidades. Ecologika cuenta con tres planes de suscripción y ofrece una prueba gratuita.

Básico

S/10 /mes

Perfecto para regiones semipobladas

  • Registro de usuario y áreas de hasta 5 hectáreas.
  • Monitoreo básico con mapas estándar
  • Reportes mensuales sobre el estado de las áreas cultivadas.
  • Alertas de deforestación dentro de un radio limitado.

Regular

S/35.85 /mes

Perfecto para regiones pobladas
